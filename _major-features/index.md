# MCdesk Major Function and Features

This section describes the **Major Function and Features** of MCdesk ITSM and how they align with **ITIL 4** practices. It is based on the *MCdesk Major Function and Features v1.0* overview.

## Mcdesk ITSM Features and ITIL 4 Practice Alignment

| Mcdesk ITSM Features | Primary ITIL 4 Practice(s) |
|----------------------|---------------------------|
| Service Desk & Incident Mgmt | Incident Management, Service Desk |
| Service Request Mgmt | Service Request Management |
| Problem Management | Problem Management |
| Change Enablement | Change Enablement |
| CMDB | Service Configuration Management |
| Knowledge Management | Knowledge Management |
| Service Level Mgmt | Service Level Management |
| Automation & Self-Service | (Guiding Principles) |
| Reporting & Analytics | Continual Improvement |

In essence, modern ITSM software is the **operational engine** that brings the ITIL 4 Service Value System (SVS) to life. It provides the structure, automation, and visibility needed to deliver value.
