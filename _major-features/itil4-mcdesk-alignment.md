# ITIL 4 Alignment – How MCdesk Relates

This page summarizes how **MCdesk ITSM** aligns with the **ITIL 4 Service Value System (SVS)**—specifically the **Four Dimensions of Service Management** and the **Guiding Principles**.

## Four Dimensions of Service Management

| Dimension | How MCdesk Relates |
|-----------|--------------------|
| **Organizations and People** | Role-based access, team assignments, and communication features. |
| **Information and Technology** | CMDB, knowledge base, and integrated discovery tools. |
| **Partners and Suppliers** | Integration capabilities for managing third-party services. |
| **Value Streams and Processes** | The software itself embodies and supports ITIL 4's value streams and processes. |

## Guiding Principles

| Guiding Principle | How MCdesk Relates |
|-------------------|--------------------|
| **Focus on Value** | Clear dashboards and reports demonstrating service value delivered. |
| **Start Where You Are** | The ability to integrate with existing tools and adapt to current processes. |
| **Progress Iteratively with Feedback** | Reporting and analytics enable iterative improvements. |
| **Collaborate and Promote Visibility** | Communication and shared dashboards foster collaboration. |
| **Think and Work Holistically** | CMDB and integration features enable a holistic view. |
| **Keep it Simple and Practical** | Intuitive interfaces and focused workflows. |
| **Optimize and Automate** | Automation and workflow features are central. |
| **Continual Improvement** | ITSM software is a critical enabler for all ITSM processes and provides the data required for ongoing improvement. |

MCdesk incident data is a key input for identifying improvement opportunities. The software helps organizations embrace ITIL 4's value-focused, holistic, and improvement-driven approach.
