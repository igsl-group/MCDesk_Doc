# 2. Service Request Management / Self-Service Portal

**Enabling Users to Request Services Smoothly, Empowering Users and Reducing Workload**

## Major Features

| Feature | Description |
|--------|-------------|
| **Service Catalog** | A user-friendly, self-service portal where users can browse and request pre-defined services (e.g., new software, hardware, access). |
| **Request Fulfillment Workflows** | Automated approval workflows (e.g., manager, finance, security). Automated, pre-defined workflows for approving and provisioning requested services. |
| **Form Builders** | Pre-defined, customizable request forms. Tools to create custom forms for service requests, collecting necessary information upfront. |
| **Approvals** | Integrated approval mechanisms for requests, ensuring proper authorization. |
| **Fulfillment tracking** | Fulfillment tracking and status updates for users. |
| **Knowledge Base Access** | Easy access for users to search and browse the knowledge base. |
| **Incident Ticket Submission & Tracking** | Users can log incidents and track the status of their requests. |
| **Status Updates & Notifications** | Users receive automated updates on their tickets. |

## ITIL 4 Process / Practice Alignment

| Practice | Alignment |
|----------|-----------|
| **Service Request Management** | The core purpose of this feature set is to manage the entire lifecycle of a service request, from submission to fulfillment. Reduces the load on the service desk by enabling users to self-resolve issues and submit requests efficiently. |
| **Service Design / Service Level Management (SLM)** | The catalog makes service offerings and their agreed-upon targets transparent. The service catalog is a direct output of service design and informs potential SLAs for requested services. |
| **Service Value Chain - Deliver and Support** | Streamlines the entire flow from request to fulfillment. |
| **Service Value System (SVS)** | Directly supports the "Co-create Value" and "Empowerment" aspects by enabling users to actively participate in service delivery. |
