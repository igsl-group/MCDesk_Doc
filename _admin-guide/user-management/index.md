# User Management

This section describes how to create and manage **Groups**, **Users**, and **Roles** in MCdesk under **System Setting** > **User Management**.

## Procedures

- [Create New Group]({{site.url}}{{site.baseurl}}/admin-guide/user-management/create-group/) – Create a group
- [Create New User]({{site.url}}{{site.baseurl}}/admin-guide/user-management/create-user/) – Create a user
- [Create New Role]({{site.url}}{{site.baseurl}}/admin-guide/user-management/create-role/) – Create a role and assign users and roles to it

For an overview of all System Setting areas, see [System Setting]({{site.url}}{{site.baseurl}}/admin-guide/system-setting/index/).
