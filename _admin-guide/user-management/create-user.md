# Create New User

To create a user in MCdesk, follow the steps below.

## Steps

1. Click on **[System Setting]** from the menu.

   ![System Setting menu]({{site.url}}{{site.baseurl}}/images/admin-guide/admin-guide-fig4.png)

2. Go to **[User Management]** > **[User]**, then click **[New]** on the top-right corner to create a user.

   ![User Management > User]({{site.url}}{{site.baseurl}}/images/admin-guide/admin-guide-fig5.png)

3. Enter the mandatory fields as below:
   - **Username**
   - **First name**
   - **Last name**
   - **Department**
   - **Password**
   - **Identity**
   - **Email**

   ![User â€“ mandatory fields]({{site.url}}{{site.baseurl}}/images/admin-guide/admin-guide-fig6.png)

4. Click **[Save]** to save the user.

After creating the user, you can assign groups, roles, and delegates as needed.
