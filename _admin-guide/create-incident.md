# Create New Incident

To create an incident in MCdesk, follow the steps below.

## Steps

1. Go to **[Incidents]** > **[Open]**.
2. Click on **[List]** from the menu.
3. Enter or select the required fields (for example, select **"Hardware"** in the **Category** field).
4. Click **[Save]** (or the appropriate button) to save the incident.

After the incident is created, it can be assigned, updated, and tracked through the normal incident lifecycle.
