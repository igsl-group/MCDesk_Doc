# Administration General

This section describes general administration procedures in MCdesk, including **Create New Incident**, **User Management** (Group, User, Role), and **System Management** (versions and license, locations, announcements). The procedures use the following notation.

## Conventions used in this guide

| Type | Format | Example | Topic |
|------|--------|---------|--------|
| **Menu navigation** | [A] > [B] | Go to [Incidents] > [Open] | Navigating from one menu to another |
| **Menu / Button / Tab** | [Square Bracket] | Click on [List] from the menu | UI elements to click |
| **Field name** | **Bold** | Select **"Hardware"** in the **Category** field | Field names in the UI |
| **Field value** | "Double quote" | Enter the value in the field | Values to enter or select |
| **Variable** | {curly_bracket} with lowercase and no space | Change {change_number} is approved by {approver} | Placeholder for a variable or dynamic value |
