# Create New Notification Template

To create a new notification template in MCdesk, follow the steps below.

## Steps

1. Click on **[System Setting]** from the menu.

   ![System Setting menu]({{site.url}}{{site.baseurl}}/images/admin-guide/notification-fig4.png)

2. Go to **[Email Settings]** > **[Notification Template]**, then click **[New]** on the top-right corner to create a new notification template.

   ![Email Settings > Notification Template]({{site.url}}{{site.baseurl}}/images/admin-guide/notification-fig5.png)

3. Enter the mandatory fields as below:
   - **Name**
   - **Table name**
4. Input the **Subject** and **Message html** which are part of the notification. Variables can be selected and inserted as needed.

   ![Notification Template â€“ Subject and Message html]({{site.url}}{{site.baseurl}}/images/admin-guide/notification-fig6.png)

5. Click **[Submit]** to save the notification template.
