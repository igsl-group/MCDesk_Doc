# Request for Service Request

Users request service requests through MyService Portal. This section describes how to navigate to the portal and submit a request.

## Navigate to MyService Portal

1. Click **[Username]** on the top-right corner and click **[Service Portal]**.
2. The MyService Portal page is shown.

## Request a Service Request

1. Click **[Quick links]** > **[Request Something]**.
2. Click the **[Service Request]** that you want.
3. Complete the form, then click **[Order Now]**.

After submission, the request follows the workflow and SLA definitions configured for that service request.
