# Request for Service Request

Users request service requests through MyService Portal. This section describes how to navigate to the portal and submit a request.

## Navigate to MyService Portal

1. Click **[Username]** on the top-right corner and click **[Service Portal]**.

   ![Username and Service Portal]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig9.png)

2. The MyService Portal page is shown.

   ![MyService Portal page]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig10.png)

## Request a Service Request

1. Click **[Quick links]** > **[Request Something]**.

   ![Quick links > Request Something]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig11.png)

2. Click one of the **[Categories]**, then click the **[Service Request]** that you want.

   ![Categories]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig12.png)

   ![Service Request selection]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig13.png)

3. Complete the form, then click **[Order Now]**.

   ![Complete form and Order Now]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig14.png)

4. Click **[Checkout]** to submit.

   ![Checkout]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig15.png)

After submission, the request follows the workflow and SLA definitions configured for that service request.
