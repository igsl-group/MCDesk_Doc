# Create a Service Request

After creating a category, create a catalog item that will be used as the service request.

## Steps

1. Click on **[System Setting]** from the menu.

   ![System Setting menu]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig4.png)

2. Go to **[Maintain Catalogs]** > **[Catalog Item]**, then click **[New]** on the top-right corner to create a new catalog item.

   ![Maintain Catalogs > Catalog Item]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig7.png)

3. Enter the mandatory fields as required by your configuration (e.g. **Category**).

   ![Catalog Item â€“ Category and fields]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig8.png)

Users can then request this service request from [MyService Portal]({{site.url}}{{site.baseurl}}/admin-guide/service-request/request-for-service-request/).
