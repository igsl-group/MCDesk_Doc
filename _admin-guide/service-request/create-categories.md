# Create a Categories

Admin needs to create a category before creating catalog items and service requests.

## Steps

1. Click on **[System Setting]** from the menu.
2. Go to **[Maintain Catalogs]** > **[Categories]**, then click **[New]** on the top-right corner to create a new category.
3. Enter the mandatory fields as required by your configuration.

After creating the category, you can use it when [creating a Service Request]({{site.url}}{{site.baseurl}}/admin-guide/service-request/create-service-request/).
