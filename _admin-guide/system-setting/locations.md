# Create Locations

To create a location in MCdesk, follow the steps below.

## Steps

1. Click on **[System Setting]** from the menu.

   ![System Setting menu]({{site.url}}{{site.baseurl}}/images/admin-guide/admin-guide-fig16.png)

2. Go to **[System Management]** > **[Locations]**, then click **[New]** on the top-right corner to create a new location.

   ![User Management > Locations]({{site.url}}{{site.baseurl}}/images/admin-guide/admin-guide-fig17.png)

3. Enter the mandatory fields as below:
   - **Name**

   ![Location â€“ Name field]({{site.url}}{{site.baseurl}}/images/admin-guide/admin-guide-fig18.png)

4. Click **[Save]** to save the location.

After creating the location, you can use it when configuring users, assets, or other records that require a location.
