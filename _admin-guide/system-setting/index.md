# System Setting

This section describes the main administration areas available under System Setting. Use these to manage users, service management, catalogs, and system configuration.

## Overview

System Setting provides access to the following areas:

### User and organization

- **User Management** – View, Create, Edit, Delete groups; assign users, other groups, roles to groups
- **Departments** – View, Create, Edit, Delete departments; assign users, other departments, cost center to departments
- **Users** – View, Create, Edit, Delete users; assign groups, roles, delegate to users
- **Roles** – View, Create, Edit, Delete roles; assign users, other roles to roles
- **Company** – View, Create, Edit, Delete company information
- **Cost center** – View, Create, Edit, Delete cost center information
- **User Management > Authentication Config**
- **User Management > Client Credentials**

### Service management

- **Service Management > SLA Definition** – View, Create, Edit, Delete SLA; change Start, Pause, Stop, Reset conditions
- **Service Management > Priority Management** – Change impact and urgency condition for different priority
- **Service Management > Dispatching Rule** – View, Create, Edit, Delete dispatching rules for auto-assigning tickets
- **Service Management > Time Sheet** – View, Create, Edit, Delete time sheets for SLA schedule

### Email

- **Email Settings > Notification Template**
- **Email Settings > Notification Rule**
- **Email Settings > Inbound Email Rule**
- **Email Settings > Sent Email History** – View logs of all email sent by system
- **Email Settings > Inbound Email History** – View logs of all email received
- **Email Settings > Email Property**

### Catalogs

- **Maintain Catalogs > Categories** – View, Create, Edit, Delete Categories
- **Maintain Catalogs > Catalog Item** – View, Create, Edit, Delete Catalog Item; assign catalog and category; assign workflow and execution plan; create and edit variables for questions; set access permission
- **Maintain Catalogs > Execution Plans** – View, Create, Edit, Delete execution plans for catalog items; create tasks when conditions are matched

### System management

- **System Management > Attachments** – View logs of all attachments from tickets
- **System Management > Access Control** – View, Create, Edit, Delete Access Control; set tables' create, delete, read, write permissions to roles
- **System Management > Versions and licenses** – View version and license information
- **System Management > Import history**
- **System Management > Locations** – View, Create, Edit, Delete Locations
- **System Management > Announcement** – View, Create, Edit, Delete Announcement; set duration and content
- **System Management > Survey Design**

### System setting

- **System Setting > Properties Config** – View, Edit Properties Config; change configuration of global properties
- **System Setting > General Config** – View, Edit General Config; change top banner, timezone, date and time format
- **System Setting > Incident Template For Portal**
- **System Setting > Incident Properties Config**
- **System Setting > Problem Properties Config**
- **System Setting > Change Properties Config**
- **System Setting > Dify Properties Config**
- **System Setting > Column Group Config**
- **System Setting > Dynamic Form Config**

### Menu and translation

- **Menu Management > Application Menu**
- **Menu Management > Page Privilege**
- **System Translation > Translated Contents**
- **System Translation > Field Labels**

For step-by-step procedures, see [Service Request]({{site.url}}{{site.baseurl}}/admin-guide/service-request/), [SLA Definition]({{site.url}}{{site.baseurl}}/admin-guide/sla-definition/), and [Workflow]({{site.url}}{{site.baseurl}}/admin-guide/workflow/).
