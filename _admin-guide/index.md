# Introduction

This document is the **MCdesk SR Module & MyService Portal Admin Guide** (Release 1.0). It describes how to administer the MCdesk ITSM system and MyService Portal.

## Purpose

The MCdesk CMDB module and MCdiscovery is implemented to provide the CMDB and CI Discovery feature for HA T team and N Team CMDB user.

## Audience

The audience for this document is for **Administrator** to admin the system. The document contains the Administration procedure required for Administrator user to operate and administrate the CMDB and Discovery features.

## MCdesk and MCdiscovery Architecture

This section describes the solution component of MCdesk and MCdiscovery.

Below are the MCdesk ITSM components:

- **Maintain Catalogs** – View all catalog that host categories and catalog item
- **Categories** – View all categories that host other categories and catalog item
- **Catalog Item** – Maintain Catalogs > Catalog Item
- **Service Portal** – For day-to-day usage, access to service request, create incident and KB
- **User tasks** – For day-to-day usage, access to the user's tasks, requests and surveys
- **Search** – Search with keywords to find useful information
- **Service Management > SLA Definition**
- **Service Management > Time Sheet** – View all time sheet for SLA definition
