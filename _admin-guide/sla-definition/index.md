# SLA Definition

This section describes the SLA (Service Level Agreement) operation in MCdesk.

## Overview

This section describes the following areas:

- **Service Management > SLA Definition**
- **Service Management > Time Sheet** – View all time sheet for SLA definition

Admin needs to create an SLA for any table where SLA tracking is required (for example, P5 incidents).

## Configure / Pre-requisite

- [Create a Time Sheet]({{site.url}}{{site.baseurl}}/admin-guide/sla-definition/create-time-sheet/) – Required before defining SLAs
- [Create SLA Definition]({{site.url}}{{site.baseurl}}/admin-guide/sla-definition/create-sla-definition/)
