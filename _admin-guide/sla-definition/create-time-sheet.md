# Create a Time Sheet

Time sheets define business hours and holidays used by SLA definitions. Create a time sheet before creating SLA definitions.

## Steps

1. Click on **[System Setting]** from the menu.

   ![System Setting menu]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig4.png)

2. Go to **[Service Management]** > **[Time Sheet]**, then click **[New]** on the top-right corner to create a new time sheet.

   ![Service Management > Time Sheet]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig16.png)

3. Enter the mandatory fields as required by your configuration (e.g. **Name**).

   ![Time Sheet – Name]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig17.png)

4. Click **[New]** to add schedule entries and configure **Repeats** as needed.

   ![Time Sheet – New and Repeats]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig18.png)

   ![Time Sheet – Repeats]({{site.url}}{{site.baseurl}}/images/admin-guide/sr-fig19.png)

After creating the time sheet, you can reference it when [creating an SLA Definition]({{site.url}}{{site.baseurl}}/admin-guide/sla-definition/create-sla-definition/).
