# Create a Time Sheet

Time sheets define business hours and holidays used by SLA definitions. Create a time sheet before creating SLA definitions.

## Steps

1. Click on **[System Setting]** from the menu.
2. Go to **[Service Management]** > **[Time Sheet]**, then click **[New]** on the top-right corner to create a new time sheet.
3. Enter the mandatory fields as required by your configuration.

After creating the time sheet, you can reference it when [creating an SLA Definition]({{site.url}}{{site.baseurl}}/admin-guide/sla-definition/create-sla-definition/).
