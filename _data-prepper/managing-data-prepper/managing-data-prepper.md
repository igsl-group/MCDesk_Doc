# Managing SmartObserve Data Prepper

You can perform administrator functions for SmartObserve Data Prepper, including system configuration, interacting with core APIs, Log4j configuration, and monitoring. You can set up peer forwarding to coordinate multiple Data Prepper nodes when using stateful aggregation.
