# Install and upgrade SmartObserve

SmartObserve and SmartObserve Dashboards are available on any compatible host that supports Docker (such as Linux, MacOS, or Windows). Additionally, you can install both products on various Linux distributions and on Windows. 

[Download SmartObserve](https://magiccreative.io/downloads.html) for your preferred platform and then choose one of the following installation guides.

| :--- | :--- |
